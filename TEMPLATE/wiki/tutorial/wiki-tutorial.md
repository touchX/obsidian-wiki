# Wiki Tutorial

完整学习 Wiki 知识库的教程。

## 学习路径

```
入门 → 进阶 → 专家
```

### Level 1: 入门 (30 分钟)

1. 阅读 [[quick-start]]
2. 创建你的第一个页面
3. 使用 Wiki 链接

### Level 2: 进阶 (1 小时)

1. 理解 [[wiki-concept]] 概念
2. 使用 Ingest 功能
3. 建立页面间的链接

### Level 3: 专家 (2 小时)

1. 分析 [[wiki-architecture]]
2. 自定义 Wiki 结构
3. 编写自己的 lint 规则
